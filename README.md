## Notebooks

### Interactive widgets

* enable

    $ jupyter nbextension enable --py widgetsnbextension --sys-prefix


### Kernels

* kernels list

    $ jupyter kernelspec list

*  add kernel

    $ ipython kernel install --user --name=projectname

* remove kernel

    $ jupyter kernelspec uninstall kernel
